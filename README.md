# BitcoinMesh LoRa Module

Bitcoin Becomes Truly Uncensorable<br>
Broadcast Bitcoin transactions over LoRa mesh networks<br>
No Internet required at the sender's location<br>

- Connection via USB or Bluetooth
- OLED Screen
- UltraLow-Power nRF52840 SoC
- HT-RA62 (SX1262) LoRa Wireless Module
- On Board LiPo Battery management
- Ready for MESHTASTIC and MESHCORE firmwares
